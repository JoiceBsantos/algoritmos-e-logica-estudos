# Matrizes (Variáveis Compostas Multidimensionais)

Exercícios desenvolvidos durante o estudo de matrizes no curso de Algoritmos e Lógica de Programação do Curso em Vídeo.

## 📚 Aulas relacionadas
- Aula 15 – "Matrizes (Variáveis Compostas Multidimensionais)"

## 🧠 Conteúdos estudados
- "Declaração de matrizes"
- "Armazenamento de dados em linhas e colunas"
- "Percorrer matrizes com estruturas de repetição"
- "Acesso a elementos por índice [linha, coluna]"
- "Diagonal principal (l = c)"
- "Operações com matrizes (soma, produto, busca de maior valor)"
- "Geração de matriz identidade"

## 💻 Exercícios implementados

1. "Leitura e exibição de matriz 3x2"
2. "Identificação e contagem de valores pares em matriz"
3. "Geração automática de matriz identidade"
4. "Operações com matriz (diagonal, produto de linha, maior valor de coluna)"

## 🎯 Objetivo

Compreender o uso de matrizes para representar dados organizados em tabelas, aplicando lógica de programação com estruturas de repetição e condicionais para resolver problemas práticos.

## 📌 Observação

Os exercícios foram desenvolvidos utilizando **Visualg**, com foco no desenvolvimento do raciocínio lógico e preparação para estruturas de dados em linguagens como **C#, Java e Python**.
