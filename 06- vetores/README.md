# Vetores

Exercícios desenvolvidos durante o estudo de variáveis compostas (vetores) no curso de Algoritmos e Lógica de Programação do Curso em Vídeo.

## 📚 Aulas relacionadas
- Aula 14 – "Vetores (Variáveis Compostas)"

## 🧠 Conteúdos estudados
- "Declaração de vetores"
- "Armazenamento de múltiplos valores"
- "Percorrer vetores com estrutura de repetição"
- "Filtros e condições em vetores"
- "Manipulação de dados em vetor"
- "Ordenação de vetores"

## 💻 Exercícios implementados

1. "Total de números pares em um vetor"
2. "Listagem de alunos com cálculo de média"
3. "Filtragem de nomes que começam com uma letra específica"
4. "Ordenação de um vetor em ordem crescente"

## 🎯 Objetivo

Compreender o uso de vetores para armazenar e manipular múltiplos dados, aplicando estruturas de repetição e condicionais para resolver problemas práticos.
