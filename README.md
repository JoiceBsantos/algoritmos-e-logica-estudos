# 💻 Algoritmos e Lógica de Programação

Repositório com exercícios e desafios desenvolvidos durante meus estudos no curso de Algoritmos e Lógica de Programação do **Curso em Vídeo**, ministrado por Gustavo Guanabara, utilizando Visualg.

---

## 📚 Estrutura do Repositório

- 📁 01-fundamentos  
  "Primeiros algoritmos, entrada e saída de dados, operadores aritméticos, lógicos e relacionais"

- 📁 02-condicionais  
  "Estruturas condicionais (Se, Senão, Escolha) e tomada de decisão"

- 📁 03-repeticao  
  "Estruturas de repetição (Enquanto, Para e Repita), contadores e acumuladores"

- 📁 04-procedimentos  
  "Criação de procedimentos, modularização e passagem de parâmetros"

- 📁 05-funcoes  
  "Funções com retorno de valor e reaproveitamento de código"

- 📁 06-vetores  
  "Armazenamento de múltiplos valores e manipulação de dados"

- 📁 07-matrizes  
  "Estruturas bidimensionais e organização de dados em tabelas"

- 📁 desafios  
  "Resolução de problemas práticos utilizando os conceitos estudados"
