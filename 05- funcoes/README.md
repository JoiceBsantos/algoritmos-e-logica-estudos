# Funções

Exercícios desenvolvidos durante a aula sobre funções no curso de Algoritmos e Lógica de Programação do Curso em Vídeo.

## 📚 Aulas relacionadas
- Aula 13 – "Funções (Retorno de valores e reutilização de código)"

## 🧠 Conteúdos estudados
- "Criação de funções"
- "Retorno de valores"
- "Passagem de parâmetros por valor"
- "Passagem de parâmetros por referência"
- "Reutilização de código"
- "Manipulação de strings"

## 💻 Exercícios implementados

1. "Função para somar dois valores (passagem por valor)"
2. "Função para verificar se um número é PAR ou ÍMPAR"
3. "Função para calcular o fatorial de um número"
4. "Função para gerar a sequência de Fibonacci (passagem por referência)"
5. "Funções de manipulação de strings no Visualg"

## 🎯 Objetivo

Compreender o uso de funções para tornar os algoritmos mais organizados, reutilizáveis e capazes de retornar valores para outras partes do programa.
